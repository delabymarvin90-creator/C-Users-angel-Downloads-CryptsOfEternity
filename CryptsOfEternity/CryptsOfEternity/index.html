Conception & Développement
Crypts of Eternity
Un roguelike top-down en pixel art généré procéduralement. Explorez des donjons infinis, affrontez des boss redoutables et forgez votre légende dans les profondeurs.
Moteur de jeu
Vanilla JavaScript
Aucun framework, aucune dépendance. Canvas 2D natif, Web Audio API pour la musique procédurale et les effets sonores.
Typographies
Google Fonts
Cinzel (titres & HUD) · Crimson Pro (texte & lore)
Fonctionnalités
Génération procédurale FOV Raycasting Web Audio API Pixel Art Canvas Système de buffs Boss fights Commerce Éclairage dynamique
Version
v4 — 2025
Crypts of Eternity est un projet personnel.
Tout le code, les sprites et la musique sont générés procéduralement en temps réel.
